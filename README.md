This repository has been developed for the target machine **Beagle Bone Black** , All the codes have been cross compiled in a Host machine.

The codes has been cross compiled for the kernel version of **6.6.21-yocto-standard preempt mod_unload ARMv7 p2v8**

# 1.simple_kernel
This project has a simple program to print hello world.

# 2.passing_arg
This project allows you to pass arguments to the kernel module and gives interrupt whenever the argument got changed.

# 3.major_minor
This project assigns the Major and Minor number for a device in static and dynamic mode.
